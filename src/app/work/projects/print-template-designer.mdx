## 项目简介

Print Template Designer 是一个基于 Web 的可视化打印模板设计器，旨在为用户提供直观、易用的模板编辑体验。

## 主要功能

- **可视化编辑**：拖拽式界面，所见即所得
- **模板管理**：创建、保存、加载模板
- **元素操作**：添加、删除、调整大小和位置
- **打印输出**：支持多种打印格式导出

## 技术栈

- React
- TypeScript
- Canvas API / SVG
